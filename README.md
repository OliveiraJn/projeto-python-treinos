
 Sistema de Planejamento de Treinos

Projeto desenvolvido em Python com o objetivo de simular um sistema de cadastro e recomendação de treinos.

-------Funcionalidades-------
- Cadastro de usuário
- Coleta de informações (nome, idade, objetivo, dias e horário)
- Cálculo de IMC
- Geração de plano personalizado

-------Tecnologias-------
- Python
- Lógica de Programação
